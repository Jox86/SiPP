Para instalar todas las dependencias necesarias:

npm install react react-dom @mui/material @mui/icons-material @mui/x-charts @emotion/react @emotion/styled xlsx html2canvas jspdf jspdf-autotable prop-types framer-motion


Instala el paquete react-icons:


npm install react-icons

Si necesitas específicamente Feather Icons (fi):

npm install react-icons --save


Core de React:

npm install react react-dom

Material UI (MUI):

npm install @mui/material @mui/icons-material @mui/x-charts @emotion/react @emotion/styled

Generación de PDF/Excel:

npm install xlsx html2canvas jspdf jspdf-autotable

Utilidades:

npm install prop-types framer-motion

TypeScript (opcional):

npm install --save-dev @types/react @types/react-dom @types/jspdf @types/html2canvas

Para desarrollo con Vite:

npm install --save-dev @vitejs/plugin-react vite

Instalar el paquete requerido:

npm install -D sass-embedded

Opcionalmente, instalar Sass normal:

npm install -D sass

instala el plugin de React para Vite:

npm install --save-dev @vitejs/plugin-react@latest

Verifica que tienes Vite instalado:

npm install --save-dev vite@latest
